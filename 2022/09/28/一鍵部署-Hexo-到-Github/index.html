一鍵部署 Hexo 到 Github

2022-09-28 255 words
hexo Github

簡單的教你如何將 Hexo 推上 Github

Step.1 建立 Github 專案

  • 登入 Github 帳號，按左上角的 New 建立新專案（repository）
  • 按進去之後，出現這個畫面，Owner不用填，然後旁邊的 Repository name 按照下圖格式填寫

    然後記得調成 Public
  • 按下 Create Repository
  • 專案建立完成～

Step.2 Hexo 部署

  • 安裝 hexo-deployer-git
    在 terminal (cmd)打上以下指令，安裝一鍵部署插件

  • 然後在原始的 _config.yml （不是 theme 底下的）修改 deploy 資料

    GitHub repository 的連結

    （就是專案建完看到的第一個可以複製的東西）

  • 改好後記得儲存

Step.3 一鍵部署

  • 輸入以下指令就建好了（\\\恭喜///）
  • 要更新頁面 g是建立靜態網頁，d是上傳到Github
